Setting Python 2.7 as the Default for Amazon Linux AMI


sudo su
yum install python27
cp -r -f /usr/lib/python2.6/site-packages/* /usr/lib/python2.7/site-packages/
cp -r -f /usr/lib64/python2.6/site-packages/* /usr/lib64/python2.7/site-packages/
rm -f /usr/bin/python /usr/bin/python2 /usr/bin/pydoc /usr/bin/pydoc2
cd /usr/bin
ln -s python2.7 python
ln -s python2.7 python2
ln -s pydoc27 pydoc
ln -s pydoc27 pydoc2

Update the python path for the easy_install command so it uses the python symlink we just set up:
